About
This is a universal solver for RTU FET RAE541 (Encoding & Encryption) course work of 2016.
Made by Linards "0x4C4A" Jukmanis.
Built using angular.js, bootstrap and uses bootswatch Simplex bootstrap theme, additionally nerdamer is used for simplifying the Mason formula equation and MathJax for rendering the LaTeX expression.
